Using the Service

  1. Pass in a url (do not include "http://" to the service.https://fierce-everglades-29355.herokuapp.com/www.google.com
  2. Receive an alternative Url in the JSON list
    {
    providedUrl: "www.google.com",
    alternativeUrl: "https://fierce-everglades-29355.herokuapp.com/1"
    }
  3. Visit the alternative URL e.g. https://fcc-learning-nicholascm90.c9users.io/1
  4. Be redirected to the original URL you passed in www.google.com
The service will not allow duplicates, and if you pass in a url including "http://", it will fail.
