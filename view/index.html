A small service providing smaller URLs

Check out the code

Using the Service

  1. Pass in a valid URL (you must include HTTP or HTTPS) https://fierce-everglades-29355.herokuapp.com/https://www.google.com
  2. Receive an alternative URL in JSON.
    {
    providedUrl: "https://www.google.com",
    alternativeUrl: "https://fierce-everglades-29355.herokuapp.com/1"
    }
  3. Visit the alternative URL e.g. https://fierce-everglades-29355.herokuapp.com/1
  4. Be redirected to the original URL you passed in https://www.google.com
The service will not allow duplicates. Invalid URLs will respond with an error.
